Élèves

Bienvenue sur ce site dont la fonction principale est de permettre à mes élèves de suivre le cours en toute circonstance. Pour cela, rendez-vous dans la section collège ou dans la section lycée et choisissez le chapitre sur lequel vous voulez travailler.

Enseignants

Vous pouvez utiliser comme bon vous semble tous les documents que vous trouverez sur ce site.

Attention cependant :

  • Le programme n'est pas encore complet, en particulier au lycée. Bien que TZR depuis plusieurs années, je n'ai pas encore eu tous les niveaux
  • Des erreurs sont encore présentes dans les documents.

Jeu de Hex

Je parle toujours un peu du jeu de Hex à mes élèves. Cela peut mener à l'organisation d'un tournois en classe ou dans l'établissement. Vous trouverez ici quelques ressources autour de ce jeu captivant.

  • Un plateau interactif codé en Javascript (cliquer sur l'image). Il n'y a pas encore d'IA mais cela viendra un jour.
  • Quelques énigmes du type "trouver le meilleur coup".
  • Des supports pour organiser un tournoi en classe.

Jeu de Hex

Le jeu de Hex est un jeu de société abstrait pour deux joueurs. Il se joue sur un plateau en forme de losange dont les cases sont hexagonales. Il fut inventé de manière indépendante par Piet Hein en 1942 et John Nash en 1948.

Cliquer sur l'image pour accèder au plateau interactif.

Choix aléatoire

Quand on est prof il arrive souvent que l'on doivent choisir un élève dans la classe. Pour cela on peut s'en remettre au hasard (mais pas toujours).

J'ai codé une application javascript permettant de choisir au hasard un élément dans une liste.

Cliquer sur le pivert pour accèder à l'application.

Jeu d'échec : chessboardjs

Chessboardjs est une application Javascript mise à disposition librement sur le site chessboardjs.com . Elle consiste en un plateau interactif permettant de positionner des pièces sur un échiquier. L'application est excellente et comme le dit l'accroche du site chessboardjs est " The easiest way to embed a chess board on your site". Néanmoins il ne s'agit en aucun cas d'une application permettant de jouer aux échecs : chessboardjs ne connaît pas les règles des échec et, a fortioris, ne comporte pas d'IA calculant le coup à jouer.

Cliquer sur la reine pour accèder au plateau intéractif
