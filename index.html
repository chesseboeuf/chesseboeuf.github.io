Cours de maths

Élèves

Bienvenue sur ce site dont la fonction principale est de permettre à mes élèves de suivre le cours en toute circonstance. Pour cela, rendez-vous dans la section collège ou dans la section lycée et choisissez le chapitre sur lequel vous voulez travailler.

Enseignants

Vous pouvez utiliser comme bon vous semble tous les documents que vous trouverez sur ce site.

Attention cependant :

  • Le programme n'est pas encore complet, en particulier au lycée. Bien que TZR depuis plusieurs années, je n'ai pas encore eu tous les niveaux
  • Des erreurs sont encore présentes dans les documents.

Jeu de Hex

Je parle toujours un peu du jeu de Hex à mes élèves. Cela peut mener à l'organisation d'un tournois en classe ou dans l'établissement. Vous trouverez ici quelques ressources autour de ce jeu captivant.

  • Un plateau interactif codé en Javascript (cliquer sur l'image). Il n'y a pas encore d'IA mais cela viendra un jour.
  • Quelques énigmes du type "trouver le meilleur coup".
  • Des supports pour organiser un tournoi en classe.

Programmation

J'utilise aussi ce site pour garder à disposition les applications que j'utilise parfois en cours.

Il y a bien sûre des applications issues du trio scratch, geogebra, python, mais aussi des applications Javascript qui ont l'avantage de pouvoir être executées partout (il suffit d'un moteur de recherche).
